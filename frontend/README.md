# RPGSoundDesk Frontend

## Run

```bash
npm install
npm run dev
```

Crie `frontend/.env` a partir de `frontend/.env.example` para configurar endpoints.

Ao abrir a aplicacao:
1. Preencha os campos de credenciais/organizacao.
2. Clique em `Registrar`.
3. Clique em `Login`.
4. Crie a mesa e opere trilhas/jogadores.
5. Use o painel `Orquestrador de Cena` para gerar audio por prompt.

## Build & Lint

```bash
npm run lint
npm run build
```
